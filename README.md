# GLE Roboflow Rest API Client
A Roboflow Rest API Client

## Installation
```shell
npm install gle-roboflow-rest-client@latest
```

## Development
Install
```
npm install
```
Build Library
```
npm run build
```
Run Tests
```
npm run test
```
